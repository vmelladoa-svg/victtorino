## INTERNO · Cómo llenar este diagnóstico

**Tiempo objetivo de preparación: 60–90 min por prospecto.** No te pases — es un anzuelo, no
una consultoría completa. El detalle fino se cobra.

### Datos a pedir / obtener antes
- URL del sitio.
- ¿Hace ads? ¿En qué plataformas? (si da acceso, mejor; si no, se evalúa lo visible).
- ¿En qué marketplaces está? (buscar su nombre/marca en ML, Falabella, Paris).

### Qué herramienta produce cada sección
| Sección | Cómo obtenerla |
|---|---|
| Web · velocidad | Medir TTFB/Core Web Vitals (PageSpeed / curl). Tu experiencia Cloudflare/Cloudways aplica directo. |
| Web · SEO | Skill `seo-audit` (audit [URL]) → da nota A–F y fixes priorizados. |
| Web · conversión/confianza | Revisión manual rápida (CTA, WhatsApp, datos legales, HTTPS). |
| Ads | Skill `ads-meta` / `ads-audit` si hay acceso; si no, inspección visible (Pixel con extensión, biblioteca de anuncios de Meta para ver sus creatividades). |
| Marketplaces | Tu conocimiento directo: buscar sus publicaciones, revisar atributos/fotos/título, comparar precio con la competencia. Esto NADIE más lo hace bien. |

### Sistema de puntaje
- Cada revisión: 🟢 = 100, 🟡 = 60, 🔴 = 20 (promedia por área).
- Puntaje global = Web ×0,35 + Ads ×0,30 + Marketplaces ×0,35.
  (Marketplaces pesa alto: es el nicho y tu diferenciador. Si el prospecto no está en ninguno,
  esa área cae a ~20 = oportunidad enorme que justifica el Pack Marketplace.)
- Nota: 90–100 A · 80–89 B · 70–79 C · 60–69 D · <60 F.

### Regla de oro
Sé **honesto y específico**. Un puntaje inflado no vende. Un hallazgo concreto ("te faltan
estos 12 atributos en estas 5 publicaciones") demuestra que sabes y construye confianza para
cerrar el retainer.
