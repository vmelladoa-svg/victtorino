### AD SET 1C — ADVANTAGE+ SHOPPING
**Presupuesto:** $1.500/día
**Formato:** Meta gestiona automáticamente (requiere catálogo de productos)
**Descripción:** Meta usa el catálogo de WooCommerce para mostrar automáticamente el producto correcto a cada persona según su comportamiento.

**Configuración:**
- Conectar catálogo de victtorino.cl → Meta Commerce Manager
- Meta optimiza creatividades, audiencias y productos automáticamente
- Solo definir presupuesto y margen mínimo aceptable (ROAS > 3x)

**Nota:** Activar después de tener 50 conversiones con pixel — Meta necesita datos para optimizar.
